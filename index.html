③ Net 비중 변화 — 패시브 스케일링 = 0

0 기준선이 패시브 스케일링(AUM이 늘어도 비중은 그대로). 양수(빨강) = 비중 순증가, 음수(파랑) = 비중 순감소. 매니저 매수와 종목별 가격 효과가 모두 합쳐진 최종 결과(Δw = w₂ − w₁). 종료일 비중 상위 15종목, 값 기준 내림차순.

④ 신규 편입 종목

기간 내 새로 등장한 종목. 전 기간 한 번도 없다가 처음 등장하면 "최초", 잠시 빠졌다 돌아오면 "재편입".

⑤ 신규 퇴출 종목

기간 내 사라진 종목. 이후로 한 번도 안 나타나면 "완전 퇴출", 다시 들어오면 "재진입함".

⑥ 보유 종목 표 (행 클릭 → 종목별 시계열)

종료일 기준 보유 종목. 행 클릭 시 해당 종목 비중·수량·평가금액 시계열 표시.

⑦ 비중 히트맵

기간 평균 비중 상위 30종목. X=날짜 / Y=종목 / 색=비중(%).

종목코드 접미사로 자동 분류 (US/KR/HK/JP/CN/Fund/Cash/Other).
