포트폴리오 KPI

선택 기간 양 끝점 기준 요약 + 시계열.

① 비중 변화 추이

기간 종료일 기준 상위 10종목 (현금 제외). 범례 클릭으로 종목 토글.
능동 Δ비중 누적 = "어제 포트폴리오를 그대로 들고 무거래였다면"의 가상 비중 대비 실제 비중의 차이를 일별로 분해해 누적한 값 (%p). 주가·환율·설정환매로 인한 변화가 제거되어 운용역의 매매만 남습니다. Net 변화는 가격 효과가 섞인 단순 비중 차이(참고용).
1종목만 활성화하면 → 그 종목의 주가(₩, 평가단가) 차트가 우측 축에 자동 오버레이 (매매-주가 타이밍 확인용). 2종목 이상 활성 시 자동 해제.

② Top Movers — 능동 매매 랭킹

능동 지표 3종의 역할 분담 — 능동 Δ비중 = 포트폴리오 내 우선순위 이동(주 지표) · 능동 매매 금액 = 절대 규모 · 매매 강도 = 그 포지션을 얼마나 공격적으로 늘렸나(무거래 가정 주수 대비, 기간 복리). 셋 모두 일별 분해 후 기간 합산이라 중간에 샀다 판 흔적도 잡습니다. Net/Gross/주수는 가격·설정환매 효과가 섞인 참고용.
연속 보유 종목만 랭킹에 표시 — 신규 편입·전량 청산은 가장 강한 신호라 아래 별도 카테고리로 분리. 설정 직후 미투입 현금의 일시 왜곡은 KPI의 현금 비중 차트로 확인하세요.

③ Net Δ비중 분해 — 능동(매매) vs 가격 효과

0 기준선이 패시브 스케일링(설정·환매로 규모가 변해도, 보유만 하면 비중 변화에 기여 0 = 무거래 상태). 최종 비중 변화 Δw = w₂ − w₁를 능동(운용역 매매) + 가격·환율 효과로 분해. 두 막대가 반대 방향이면 "오르는데 팔았다 / 빠지는데 샀다"는 의미. 종료일 비중 상위 15종목, 능동 기여 내림차순.

④ 연속 매집 · 연속 매도 스트릭

일별 능동 변화가 같은 방향으로 3거래일 이상 연속된 구간. 하루의 큰 매수보다 "여러 날 연속 소량 매수"가 운용역 확신의 신호로 더 유의미합니다. 기간 종료일까지 이어지면 진행중으로 표시.

⑤ 신규 편입 종목

기간 내 새로 등장한 종목. 전 기간 한 번도 없다가 처음 등장하면 "최초", 잠시 빠졌다 돌아오면 "재편입".

⑥ 신규 퇴출 종목

기간 내 사라진 종목. 이후로 한 번도 안 나타나면 "완전 퇴출", 다시 들어오면 "재진입함".

⑦ 보유 종목 표 (행 클릭 → 종목별 시계열)

종료일 기준 보유 종목. 행 클릭 시 해당 종목 비중·수량·평가금액 시계열 + 일별 능동 매매(추정 체결단가 포함) 표시.
편입 후 추정 수익률 = 현재 보유 사이클(마지막 편입일 이후)의 능동 매수 가중 평균단가 대비 현재 내재가(원화 환산·분할 조정). * = 데이터 시작 전부터 보유 → 데이터 시작일 이후 기준.

⑧ 종목별 매매 궤적 (현재 보유 + 과거 청산 전 종목)

현지통화 수정주가(yfinance)에 매니저의 능동 매수·매도를 겹쳐 봅니다. 능동 %p = 비중변화에서 가격효과·펀드 자금유출입(패시브 스케일링)을 제거한 순수 매니저 액션. 평균단가는 능동 매수가 일어난 가격의 가중평균(추정). 점이 클수록 큰 능동 매매.

가격 데이터가 연계된 종목이 없습니다. fetch_prices.py → build_trajectory.py 실행 후 새로고침하세요.

⑨ 비중 히트맵

기간 평균 비중 상위 30종목. X=날짜 / Y=종목 / 색=비중(%).

⑩ 테마별 비중 추이 (일별)

종목→테마 매핑(themes.js, 전 펀드 공용 — 수정 후 새로고침만 하면 반영) 기준으로 일별 공시 비중을 테마 단위로 합산. 펀드의 테마 로테이션(갈아타기)이 한눈에 보입니다. 차트에 커서를 올리면 해당 날짜의 전체 테마 구성이 표시됩니다.
